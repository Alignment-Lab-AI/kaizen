Follow these steps to generate tests for your website.

1. Update the URL in the file at `examples/e2e_test/generate.py` - Line 5.

2. Run the following command to generate tests.
```bash copy
    PYTHONPATH=. poetry run python examples/e2e_test/generate.py
```
  
Kaizen will generate all the tests and store them inside `.kaizen/tests/`

Follow these steps to generate ui-tests for your website.

1. Update the URL in the file at `examples/e2e_test/generate.py`.

2. Run the following command to generate tests.

```bash
    PYTHONPATH=. 
    poetry run python examples/e2e_test/generate.py
```
  
Kaizen will generate all the tests and store them inside `.kaizen/ui-tests/`

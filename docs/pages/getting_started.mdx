You can start using Kaizen in two ways:

## Quickstart 
Use our web [app](https://kaizen.cloudcode.ai) to get started quickly. No setup needed - just log in and go. Perfect for trying out Kaizen or for teams who want a simple, ready-to-use solution with a user-friendly interface.
 
## Self-Hosted Deployment
Deploy your own instance of the open-source Kaizen project by following the installation instructions provided in our documentation. Follow the steps outlined in the self-hosting installation [guide](self_hosting_guide) to set up Kaizen on your own infrastructure.

## Step 1: Sign Up for an Account

1. Visit [https://kaizen.cloudcode.ai](https://kaizen.cloudcode.ai)
2. Click on the "Sign Up" button in the top right corner
3. Fill out the registration form with your details
4. Verify your email address by clicking the link sent to your inbox

## Step 2: Create a New Project

Once you're logged in:

1. Click on the "New Project" button on your dashboard
2. Choose a name for your project
3. Select the repository you want to connect (GitHub, GitLab, or Bitbucket)
4. Follow the prompts to authorize Kaizen to access your repository

## Step 3: Configure Kaizen bot 

Install the kaizen bot by going to this link: [![Install Kaizen App](https://img.shields.io/badge/Get%20Kaizen%20App-Install-8A2BE2)](https://github.com/apps/kaizen-bot)
1. Check the permission which kaizen bot requires.
2. Select repositories you want to install kaizen bot.


## Next Steps

Kaizen bot should start working whenever you create a PR.
